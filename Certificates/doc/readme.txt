1. Запуск приложения. 
- 1.1 Для запуска приложения из папки client выполнить npm start. Адрес при запуске локально: http://127.0.0.1:3002/
- 1.2 Для запуска приложения и дальнейшего отслеживания изменений выполнить из папки client:
- - npm run dev: отслеживает изменения клиента
- - npm run server: отслеживает изменения сервера
