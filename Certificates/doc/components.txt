Структура компонентов приложения в папке client/src:

index.tsx. Главный react-компонент, производит рендер остальных

- assets. Здесь находятся ресурсы: 
- - fonts. Шрифты
- - images. Изображения 

- shared. Здесь находятся локализованные файлы, доступные для любых react-компонентов: 
- - types. Типы TypeScript 
- - utils. Утилиты 

- components. Здесь находятся react-компоненты:
- - Main. Компонент верхнего уровня, производит запрос на сервер за данными сертификатов. Контролирует переключение роутов
- - List. Родительский компонент для списка сертификатов
- - - Certificates. Список сертификатов
- - Detailed. Детальная страница просмотра сертификата. Ищет сертификат в ранее загруженном с сервера списке по параметру из URL
